Nuti Eenyu?

Waayee Keenya: Kaayyoon keenya lafa gabaa sararraa hunda-galeessa, uffata qulqullinni isaa eegame dhiyeessu uumuudha. Seenaa keenya kan jalqabame bara 2021, gosa tajaajila hunda-galeessa, fi hanga biyyaatti mul’achuu fi fedhii ummataa guutuuf kan xiyyeeffanne dha. Nuti garee hojjattoota gootota, dhaloota haaraa fi tajaajila keenya cimsuuf yaada fooyyee qabaachuudhaan hojjannu qabna.

Seenaa Keenya:

Dhaabbanni keenya bara 21 kan eegalame yoo ta’u, hanga har’aatti bifa foyya’aa fi tajaajila gaarii kennuudhaan beekkamaa dha. Nuti ni amanna tajaajila keenya fooyyessuu fi gaarii kan taasisuu hordoftoota keenya bira bu’uun. Kaayyoon keenya bara dabrerraa jalqabee gaaffii fi fedhii ummataa guutuu dha.


Garee Keenya:

Gareen keenya kan ijaaramu namoota danda’a qabeessa, hojii isaanii irratti amanamoo fi haala hojii isaanii fooyyessuu barbaadanii dha. Nutis hojjattoota keenya qajeelcha kennuudhaan fi dinqisiifatamuudhaan deeggarra.


Maatiin Keenya:

Maatiin keenya tajaajila keenya biratti dinqisiifataa fi deeggartoota keenya giddugaleessa godhuu irratti. Nuti hojjattoota keenya gorsa gaarii kennuudhaan fi taayitaa kennuudhaan dinqisiifanna.
